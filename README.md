
# Mini Transformer Benchmark

## Description
This project implements a mini Transformer encoder from scratch for a sequence classification task.

## Task
Predict whether the first non-padding token appears in the second half of the sequence.

## How to run

Train:
python train.py

Benchmark:
python benchmark.py

## Model
Includes:
- Embedding
- Positional Encoding
- Multi-head Attention
- Feed-forward Network
- Transformer Encoder

## Restrictions
No built-in Transformer libraries are used.
